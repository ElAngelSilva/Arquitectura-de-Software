| :---- | :---- |
| **Proyecto** | FINTECH & MICRO-PRÉSTAMOS |
| **Atributo** | Traspaso de dinero entre usuarios/cuentas, y la validación de las mismas |
| **Justificación** | La importancia de validar diferentes variables a la hora de hacer transferencias (saldo disponible en la cuenta que está transfiriendo, datos de la cuenta que recibe, conexión, etc) |
| **Escenario** | "Cuando un usuario intenta transferir a una cuenta, pero alguno datos de la cuenta destina esta mal (**Estímulo**), el sistema debe bloquear la transacción (**Respuesta**), haciendo que el 100% del dinero regrese a la cuenta origen (**Métrica**)". |
